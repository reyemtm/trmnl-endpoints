# Endpoints

A collection of focused endpoint functions for trmnl display plugins.

> https://trmnl.getbounds.dev

## Packages

- [`@endpoints/nws-forecast`](./packages/nws-forecast) - National Weather Service forecast data wrapper

## Installation

Install individual packages from this monorepo:

```bash
npm install github:reyemtm/trmnl-endpoints#subdirectory=packages/nws-forecast
```

## Development

```bash
npm install
npm test
```
